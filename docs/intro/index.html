チュートリアル - はじめに

Piwigoは、Web上で多数の画像が閲覧できるフォトギャラリーシステムです。

Piwigoの画像管理の基本操作を５分間で理解するチュートリアルを始めます。

Piwigoのページ構成​

Piwigoには、画像を閲覧するためのフロント画面と画像を管理するための管理画面があります。

フロント画面​

トップ画面から アルバム一覧 ⇒ 画像一覧 ⇒ 画像詳細 と画面遷移します。

⇒ ⇒

管理画面​

管理画面の左サイドには管理機能メニューがあります。
管理機能を表示するにはログインが必要です。

アルバム​

画像をまとめる単位がアルバムです。画像ファイルをフォルダ的に収める入れ物です。
アルバムの中にアルバムを配置して、アルバムの階層構造をつくることもできます。
