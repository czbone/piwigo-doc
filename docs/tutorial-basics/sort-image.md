# 画像の並び順を変更

アルバム内の画像の並び順を変更します。

![画像一覧](/img/sort_image1.png)

## 管理画面にログイン

[**ログイン**](/docs/tutorial-basics/login-logout.md)を行って、管理画面を表示します。

## アルバム管理画面を表示

**管理機能メニュー**の「**アルバム>管理**」をクリックし、アルバム管理画面に遷移します。
